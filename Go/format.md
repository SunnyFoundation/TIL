# Format 

* %v
* %T



### %v
```go

package main
import "fmt"
func main() {
    var conferenceName = "Go Conference"
  
    fmt.Printf("Welcome to %v booking Applicaiton\n", conferenceName)

}

```



### %T
```go

package main
import "fmt"
func main() {

  
       var conferenceName = "Go Conference"

    fmt.Printf("conferenceName Type is  %T \n", conferenceName)


}



```
