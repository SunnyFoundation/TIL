
# bisect

```bash

when you found bug in your code
git help you to found bug using Binary Search

```



### Binary Start


```bash

when you found bug in your code
git help you to found bug using Binary Search

```


### Mark the Buggy Point




```bash
git bisect bad

```





### Move to a Suspected Point





```bash
git checkout (해당 커밋 해시)

```









### Mark as Good if No Bug





```bash
git bisect good


```






### Mark as Bad if Bug Still There






```bash
git bisect bad 



```




### End Binary Search








```bash
git bisect reset




```
