# Stash



```bash
You are coding very hard but  suddenly your boss or your client request something to you
So you have to change your code immediately
 then now the code that you write save stash area and If Finish your task and take it out and rewrite it 



```
## Save your code


```bash
git stash

```


## Take it out of stash area 



```bash

git stash pop
```
