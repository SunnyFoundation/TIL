# Primitive Type 


 ⚡️ It could store only one value 





* ` number
`


``` typescript 
let num : number = 10; 
```



* ` String
`


``` typescript 
let str : string = "Hello, TypeScript!";
```



* ` Boolean
`


``` typescript 
let isTrue : boolean = true;
```



* ` Undefined
`


``` typescript 
let undefinedValue : undefined = undefined;
```





* ` Literal type 
`


``` typescript 
let literalNum: 42 = 42;
let literalStr: "Hello" = "Hello";
let literalBool: true = true;

```
